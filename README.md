IMEX — Real-Time Supply Chain Intelligence Dashboard
IMEX is a supply chain risk analysis platform designed to give logistics teams and operational leaders real-time visibility into global supply chain disruptions, financial exposure, and risk trends.

## Key Features
* Real-Time KPI Dashboard: Instant access to critical metrics including Active Disruptions, Overall Risk Score, Revenue Exposure ($), and Active Alerts.

* Risk Trend Analytics: Interactive visual charts tracking time-series risk trends to help teams anticipate potential bottlenecks.

* Live Disruptions Feed: Contextual event updates detailing severe real-world events (e.g., port closures, supplier strikes, weather warnings).

* Dependency Graph & Vulnerability Analysis: Dedicated views to visualize complex supply chain dependencies and evaluate key risk factors.

* Automated Executive Reports: One-click generation of PDF summaries for quick executive reporting.

## Tech Stack
* Frontend
   Framework: React / Next.js with TypeScript

   Styling: CSS / Tailwind CSS

    Hosting: Render

* Backend & Analytics
  Core Logic: Python

   Deployment: Docker / Dockerfile

## Getting Started
Prerequisites
Make sure you have Node.js and Python installed on your machine.

Node.js (v18+ recommended)

Python (v3.10+ recommended)

## Installation
1. Clone the repository:

```Bash
git clone https://github.com/Abhilash-42/IMEX.git
cd IMEX
```

2. Setup and run the backend:

```Bash
#Navigate to your backend directory and install dependencies
pip install -r requirements.txt
python main.py
```

3.Setup and run the frontend:

```Bash
#Navigate to your frontend directory
cd imex-main
npm install
npm run dev
```

4. Access the application:

Open your browser and navigate to http://localhost:3000.
